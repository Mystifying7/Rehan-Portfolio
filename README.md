# 🚀 Rehan Portfolio

A modern, responsive developer portfolio built with **React + Vite** showcasing my projects, technical skills, certifications, hackathons, and journey as a Computer Science student focused on Artificial Intelligence and Frontend Development.

## 🌐 Live Website

👉 https://rehanportfolio-chi.vercel.app/

---

## ✨ Features

- Modern Hero Section
- Interactive Particle Background
- Aurora Background Animation
- Responsive Navigation
- Skills Showcase
- Project Gallery
- Hackathon Experience
- Education Timeline
- Certifications
- Resume Download
- Smooth Animations using Framer Motion
- Mobile Responsive Design

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Mystifying7/Rehan-Portfolio.git
```

Go inside the project

```bash
cd REHAN-PORTFOLIO
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
